Password Protection

Password protecting my webpage was a bit tricky as the website host I'm using does not support htaccess which is the most commonly used method. Since github pages only supports static pages I found a solution online that encrypts the whole html file, the browser then decrypts it using the password provided. This method is not the sturdiest of methods but I'm sure it'll be fine for the purposes of this course.

New Tag Used

The new tag that I used for this 1st lab was the button tag. It simply creates an html button on the web page, that offers the user a way to directly interact with the website. When the button is clicked on my homepage, a broswer alert pops up and notifies the user that the button was clicked.

Browser Differences

The website for the most part looks the same on different browsers. The only real difference I noticed was that the alert box that pops up when the user clicks my button, looks a bit different depending on what browser is used.
