Lab 2 Technical Report

Element Classes

I used two element css tags. They are; div.card, which has styling to create a modern card design on top of the background, as well as h1.title to create a different and unique title at the top of the page.

Global CSS Classes

I used several global css tags. The main ones are related to the nav bar at the top of the page. These classes include; .active which is used to change the color of the menu item at the top, as well as .nav-bar & .nav-items which include several styles to transform content into a navigational menu style.

HTML 5-only Tags

The two HTML 5 tags I used are the details and progress tags. Both can be found on the homepage.

CSS3-only Features

The two CSS3 only features that I used include; the linear gradient function in background-image and the box-shadow style. Both of these are used in the background on this page.

Password Protection

The password protection method I used is the same as in lab 1. I found a solution online called StatiCrypt which works with githubs static page setup. This program, based on a crypto-js library, encrypts the whole html file using AES-256. The password protected page is then decrypted in-browser, using the password provided.
